# Quiz Game - Flutter
A simple and fun quiz game built with Flutter, featuring multiple-choice questions with images, score tracking, and a restart option.
Features

Display questions with images
True/False answer buttons
Visual feedback (green thumbs up / red thumbs down)
Score tracking
Game over alert with results and restart option
Reset button to start over
Clean and responsive UI

# How to Run

Clone the repository:Bashgit clone https://github.com/Dhia-Khayati/Flutter_Gess_app.git
cd Flutter_Gess_app
Install dependencies:Bashflutter pub get
Run the app:Bashflutter run

# Requirements

Flutter SDK (latest stable version)
Dart SDK

# Project Structure

lib/main.dart: Main application and game logic
lib/question.dart: Question class model
lib/question_brain.dart: Question data and brain logic
images/: Folder containing question images
